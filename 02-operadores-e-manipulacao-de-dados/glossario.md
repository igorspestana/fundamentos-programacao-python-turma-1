# Glossário Resumido — Aula 2

### Reatribuição

Atribuir um novo valor a uma variável que já existe. Ex: `idade = 31` e depois `idade = 32`.

### Tipo de dado

Classificação formal que define a natureza de um dado: quais valores pode assumir, como é armazenado e quais operações são permitidas.

### `float`

Tipo para números decimais. Ex: `19.90`, `1.75`, `3.14`.

### `bool` (booleano)

Tipo que representa verdadeiro ou falso: `True` ou `False`. Resultado de comparações e lógica.

### `type()`

Função que retorna o tipo de um valor. Ex: `type(idade)` retorna `<class 'int'>`.

### Casting (conversão de tipos)

Transformar um valor de um tipo em outro usando `int()`, `float()` ou `str()`.

### `int()`, `float()`, `str()`

Funções que convertem valores entre tipos. Ex: `int("31")` vira `31`, `str(42)` vira `"42"`.

### Concatenação

Juntar strings com o operador `+`. Ex: `"10" + "20"` resulta em `"1020"`.

### `ValueError`

Erro que ocorre quando uma conversão é inválida (ex: `int("abc")`).

### Operadores aritméticos

Símbolos para operações matemáticas: `+`, `-`, `*`, `/`, `//`, `%`, `**`.

### Divisão inteira (`//`)

Divisão que descarta a parte decimal. Ex: `10 // 3` resulta em `3`.

### Módulo (`%`)

Operador que retorna o resto da divisão. Ex: `10 % 3` resulta em `1`.

### Potenciação (`**`)

Elevar um número a uma potência. Ex: `2 ** 3` resulta em `8`.

### Precedência de operadores

Ordem em que o Python executa as operações: parênteses primeiro, depois `**`, depois `*`/`/`/`//`/`%`, depois `+`/`-`.

### Operadores de atribuição compostos

Forma abreviada de atualizar uma variável: `+=`, `-=`, `*=`, `/=`. Ex: `x += 1` equivale a `x = x + 1`.

### Operadores de comparação (relacionais)

Retornam `True` ou `False`: `==` (igual), `!=` (diferente), `>`, `>=`, `<`, `<=`.

### Operadores lógicos

Combinam condições booleanas: `and` (as duas verdadeiras), `or` (pelo menos uma verdadeira), `not` (inverte).

### Condição

Expressão que resulta em `True` ou `False`. Usada para o programa decidir o que fazer.

### `input()`

Função que pausa o programa e espera o usuário digitar algo. Sempre retorna texto (`str`).

### f-string

Formato de string com variáveis entre chaves. Ex: `f"Soma: {a + b}"`.
